# HuaweiSMSender
Python script to send SMS using Huawei new[er] USB dongles via the HTTP interface

# Compatible devices - tested
Huawei E3372

# Compatible devices - should work
Huawei E3131
Huawei E3131

# Prerequisites

## Mandatory
python-requests

## Recommended
python-bs4
